# 🎥 Воркшоп по Claude Code

Юра [@yosakohf](https://t.me/yosakohf) провёл лекцию по практике использования Claude Code в коммерческих проектах.

Юра работает GoLang бэкендером, также писал на Python и Java.

[Запись воркшопа](https://drive.google.com/drive/folders/1lVSU7LY7l_mQCPaLmUuxLiBkNm5ZCI5y)  
[Материалы](https://docs.google.com/document/d/1Ufob9EVI_tQuBF-tIu3ZPt6gZK-AZcx0lj05rU5cXVg)

### О чём поговорили

- AI-автодополнение, чаты и агенты
- Кейсы использования агентов с демо: написание нового кода, решение проблем, покрытие кода тестами, рефакторинг
- Управление контекстом AI-агента
- Конфигурирование Claude Code: правила, хуки, навыки, MCP
- Саб-агенты

Воркшоп будет полезен тем, кто хочет понять, как пользоваться AI-агентами в реальной работе.
